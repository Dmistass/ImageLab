# ImageLab

Програмата ImageLab позволява прилагането на филтри към изображения във формат NETpbm (.ppm, .pgm, .pbm).

Може да чете всеки тип NETpbm.
ImageLab винаги запазва изображенията в бинарен вид – най-оптимизирания.

Налични филтри:
  - Замъгляване
  - Контур
  - Контраст
  - Нормализиран контраст
  - Негатив
  - Изостряне
  - Праг

Налични команди:
| Команда | Описание |
|-------------------------------|---------------------------------|
| load <път1> <път2> | Зарежда изображения от файлове |
| remove <име> | Премахва изображение |
| printAll | Показва всички заредени изображения |
| add-filter <изображение> <филтър> | Добавя филтър към изображение |
| remove-filter <изображение> <индекс> | Премахва филтър по индекс |
| show-filters <изображение> | Показва филтрите на изображението |
| show-all-filters | Показва филтрите на всички изображения |
| run <изображение> | Прилага филтрите към изображението |
| run-all | Прилага филтрите към всички изображения |
| save <изображение> [път] | Запазва изображение |
| show-available-filters | Показва всички налични имена на филтри |
| help | Показва съобщение за помощ |
| quit | Изход от програмата |
